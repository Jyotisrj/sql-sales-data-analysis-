# SQL Sales Data Analysis Project

## Objective
Analyze retail sales data using SQL to generate business insights such as total revenue, customer spending patterns, and product performance.

## Tools Used
- SQL (MySQL)

## Steps Performed
- Created sales database and tables
- Inserted transaction dataset
- Performed filtering and aggregation
- Generated insights using GROUP BY and aggregate functions

## Key Insights
- Calculated total sales revenue
- Identified top-performing products
- Analyzed customer spending trends
