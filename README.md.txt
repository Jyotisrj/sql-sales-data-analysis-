# SQL Retail Sales Analysis Project

## Project Overview
This project demonstrates SQL-based data analysis using customer and sales datasets.
The goal is to extract insights such as customer spending patterns and city-wise sales.

## Key SQL Concepts Used
- SELECT statements for data extraction
- WHERE clause for filtering
- JOIN operations between multiple tables
- GROUP BY for aggregation
- Handling NULL values using COALESCE
- Removing duplicate records

## Tools Used
SQL (MySQL)

## Insights Generated
- Customer-wise total spending
- City-wise sales trends
- Product sales distribution
